<h2>JSON Formatter</h2>

<p>
JSON Formatter is a modern, browser-based utility designed to help developers, testers,
business analysts, API engineers, and data professionals efficiently format, validate,
analyze, and manage JSON data. The application operates entirely on the client side,
requiring no server, backend services, or external dependencies, making it secure,
fast, and fully functional offline.
</p>

<h3>Project Overview</h3>

<p>
The tool provides an intuitive dual-panel interface where users can paste raw JSON
input and instantly transform it into a structured, readable format. It helps improve
data readability, simplifies debugging, and allows users to quickly identify JSON
syntax issues through real-time validation and error reporting.
</p>

<h3>Key Features</h3>

<ul>
<li><strong>JSON Beautification</strong> – Converts compact JSON into a properly indented and readable format.</li>

<li><strong>JSON Minification</strong> – Removes unnecessary spaces and line breaks to generate compact JSON output.</li>

<li><strong>JSON Validation</strong> – Verifies JSON syntax and displays detailed error messages when invalid data is detected.</li>

<li><strong>Syntax Highlighting</strong> – Applies color coding to JSON keys, strings, numbers, booleans, null values, and structural elements for enhanced readability.</li>

<li><strong>Search Functionality</strong> – Allows users to search and highlight specific values, keys, or text within formatted JSON output.</li>

<li><strong>JSON Statistics Dashboard</strong> – Displays useful metrics such as:
    <ul>
        <li>Total Characters</li>
        <li>Total Lines</li>
        <li>Number of Objects</li>
        <li>Number of Arrays</li>
        <li>Total Keys</li>
    </ul>
</li>

<li><strong>Copy to Clipboard</strong> – Enables quick copying of formatted JSON for reuse in other tools and applications.</li>

<li><strong>Import JSON Files</strong> – Supports loading local JSON files directly into the editor.</li>

<li><strong>Export JSON Files</strong> – Allows users to download beautified or minified JSON output.</li>

<li><strong>Dark and Light Themes</strong> – Provides theme switching with automatic preference persistence using local storage.</li>

<li><strong>Responsive Design</strong> – Optimized for desktop, tablet, and mobile devices.</li>

<li><strong>Offline Operation</strong> – Works entirely within the browser without requiring internet access.</li>
</ul>

<h3>User Interface</h3>

<p>
The application features a clean and modern interface consisting of:
</p>

<ul>
<li><strong>Top Toolbar</strong> containing all primary actions such as Beautify, Minify, Copy, Import, Export, Clear, and Theme Toggle.</li>

<li><strong>Status Notification Area</strong> displaying validation success and error messages.</li>

<li><strong>Search Bar</strong> for locating content within JSON output.</li>

<li><strong>Statistics Panel</strong> providing real-time JSON metrics.</li>

<li><strong>Input Editor</strong> for entering or loading JSON content.</li>

<li><strong>Output Viewer</strong> displaying formatted and syntax-highlighted JSON.</li>

<li><strong>Toast Notifications</strong> for user feedback during operations.</li>
</ul>

<h3>Technical Highlights</h3>

<ul>
<li>Built using pure HTML, CSS, and Vanilla JavaScript.</li>
<li>No frameworks or external libraries required.</li>
<li>Client-side JSON parsing and validation.</li>
<li>Local storage support for theme persistence.</li>
<li>Custom syntax highlighting engine.</li>
<li>Dynamic JSON object and array analysis.</li>
<li>Fully self-contained single-file architecture.</li>
</ul>

<h3>Use Cases</h3>

<ul>
<li>API response formatting and validation.</li>
<li>Debugging JSON payloads.</li>
<li>REST API development and testing.</li>
<li>Data transformation and inspection.</li>
<li>Learning and understanding JSON structures.</li>
<li>Preparing JSON configurations for applications and services.</li>
</ul>

<h3>Conclusion</h3>

<p>
JSON Formatter is a lightweight yet powerful productivity tool that simplifies JSON
editing, validation, formatting, and analysis. Its intuitive design, real-time

</p>
